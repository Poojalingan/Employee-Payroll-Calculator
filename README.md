# 💼 Employee Payroll Calculator

## 📌 Overview

A modern web-based payroll calculator built using AngularJS and Bootstrap. It provides a clean UI to calculate gross salary, tax deduction, and net salary.

## 🚀 Features

* Responsive and modern UI
* Calculates Gross, Tax, and Net Salary
* Dynamic result display
* Handles user input efficiently

## 🛠️ Tech Stack

HTML | CSS (Bootstrap 5) | JavaScript | AngularJS

## ▶️ Usage

Open the project in a browser, enter employee details, and click **Calculate** to view salary breakdown.
